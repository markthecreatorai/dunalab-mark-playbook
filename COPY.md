**1) Isso serve para quem não sabe programar?**
Sim. O foco é operação e estrutura. Código é opcional.

**2) Isso é só prompt?**
Não. Prompt é só um detalhe. O núcleo é identidade, memória, rotina e governança.

**3) Quanto tempo para colocar no ar?**
Com foco: uma tarde para base funcional.

**4) Posso usar com qualquer modelo?**
Sim, desde que você mantenha as regras de segurança e acesso.

**5) Tem risco de vazamento?**
Toda automação tem risco. O playbook ensina como reduzir isso com canais autenticados, mínimo privilégio e confirmações.
